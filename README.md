- TypeScript
- Styled Components
- React Navigation
- React Hook Form
- Yup
- Axios


## Para baixar o projeto:

Clone o repositório

```
  git clone https://github.com/talles-morais/MangoHQShop-Mobile.git
```

Entre no diretório do projeto

```bash
  cd MangoHQShop-mobile
```

Instale as dependências do node
```
  npm install
```

Inicie o app com o expo
```
  npm run start
```
